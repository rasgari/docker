# نصب و راه‌اندازی

مقدمات

نسخه ۸.۱۸.۳ این محصول همراه با رفع چندین آسیب‌پذیری منتشر شده است. 
پیش‌نیاز: زیرساخت مجازی (مثلاً vSphere)، منابع مناسب سخت‌افزاری، DNS و NTP هماهنگ. 
نصب معمولاً با استفاده از OVA یا ابزار VMware Aria Suite Lifecycle انجام می‌شود.

مراحل نصب

فایل OVA یا ایمیج مربوطه را از پورتال VMware/Broadcom دانلود و در vSphere ایمپورت کنید.
تنظیمات اولیه شامل: نام هاست، IP ثابت، گیت‌وی، DNS و NTP. 
راه‌اندازی کلستر (مثلاً یک مستر و چند Node کارگر) برای تولید، ذخیره و تحلیل لاگ‌ها. 
پس از نصب، از طریق UI وارد شوید و مجوز (license) را فعال کنید، و چک سلامت سرویس‌ها داشته باشید.
پورت‌های پیش‌فرض و سرویس‌های مرتبط را بررسی کنید و اطمینان حاصل کنید که فایروال‌ها و شبکه اجازهٔ ارتباطات لازم را دهند.


نکات مهم

حتماً Snapshot از VM قبل از هر ارتقاء بگیرید.
بررسی کنید نسخه CPU, RAM, Storage مناسب باشد بر اساس حجم لاگ‌ها.
از DNS با رکورد معکوس و NTP هماهنگ استفاده کنید تا تحلیل و گزارش به مشکل نخورد.
